FLAP Discord Rich Presence

This companion is optional. Start Discord normally, boot FLAP in VirtualBox,
then run FLAP-Rich-Presence.exe on the host machine.

The FLAP Discord Application ID is already built into the executable. An
Application ID is a public Rich Presence identifier and cannot be used to log
in to Discord or control the application. The executable does not contain a
client secret, bot token, OAuth token, webhook, password, or Discord login.

The companion only connects to FLAP through CARROT on the local machine. Close
the executable to disable rich presence.
